# Security Policy

## Supported Scope

This repository contains a data conversion pipeline and generated Markdown law files. Security reports should focus on the converter, GitHub Actions workflow, repository automation, or any issue that could compromise generated output integrity.

## Reporting A Vulnerability

Please open a private security advisory on GitHub if possible. If that is not available, contact the maintainer through the GitHub profile linked from this repository.

Please include:

- A short description of the vulnerability.
- Steps to reproduce.
- Impact on generated law files, repository automation, or users.
- Suggested mitigation, if known.

## Data Integrity

If you find incorrect law text, stale data, or formatting that changes legal meaning, open a normal issue unless there is an active exploitation or supply-chain risk. The official National Laws and Regulations Database remains the authoritative source.
